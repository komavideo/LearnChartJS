Chart.js安装
=============

## 知识点

* Chart.js的安装方法

## 安装方法

### npm安装

~~~bash
npm install chart.js --save
~~~

### bower安装

https://bower.io/

~~~bash
bower install chart.js --save
~~~

### CDN

https://cdnjs.com/libraries/Chart.js

### 下载本地安装（本课程采用）

https://github.com/chartjs/Chart.js

建立myweb文件夹，将下载的chart.js文件拷贝进来，然后在网页中引用。
